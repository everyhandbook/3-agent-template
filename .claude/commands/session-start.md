# Session Start Workflow

이 워크플로우는 새로운 AI 세션을 시작할 때 컨텍스트를 파악하기 위해 실행합니다.

## 1. 세션 로그 확인
- `docs/session-logs/` 디렉토리의 가장 최신 로그 파일을 읽습니다.
- 이전 세션에서 남긴 `Next Steps`와 `In Progress` 항목을 확인합니다.

## 2. 작업 상태 확인
- `task.md`를 읽어 현재 진행 중인 Task와 전체 진행률을 파악합니다.
- `README.md`를 읽어 프로젝트의 전체 목표와 현재 Phase를 상기합니다.

## 3. 사용자 확인
- 사용자에게 "이전 세션의 [작업 내용]에 이어 [다음 단계]를 진행하시겠습니까?"라고 확인합니다.
