# /checkpoint 워크플로우

세션 전체를 종료할 정도는 아니지만, 현재 작업 내용을 레포에 정리하고 작은 단위의 커밋으로 남기고 싶을 때 사용합니다.

## 수행 절차

1. 이번 checkpoint 범위를 짧게 정리합니다.
2. 관련 파일만 업데이트합니다.
3. `.claude` 변경이 있으면 `.agent` 미러까지 sync합니다.
4. 관련 파일만 stage하고 커밋합니다.
5. 커밋 해시와 포함 파일, `session-wrap-up` 필요 여부를 안내합니다.

## 주의사항

- unrelated 변경은 함께 커밋하지 않습니다.
- push는 별도 요청 시에만 진행합니다.
