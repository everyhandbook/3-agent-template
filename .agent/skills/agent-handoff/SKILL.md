# Agent Handoff

Prepare a handoff that lets another agent or a later session continue work without rebuilding context from scratch.

## Structure

1. Goal of the work
2. Current status
3. Changed files or related documents
4. Remaining work
5. Risks and checks

## Preferred Location

- Record a small handoff in `docs/session-logs/`
- Use a separate log file for larger transitions
