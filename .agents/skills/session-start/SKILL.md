# session-start

1. `git status`로 미커밋 변경사항을 확인합니다.
2. `task.md`를 읽고 현재 단계와 다음 할 일을 확인합니다.
3. `docs/session-logs/` 최신 로그를 확인합니다.
4. `README.md`, `AGENTS.md`를 읽고 운영 규칙을 상기합니다.
5. 필요하면 `docs/lessons.md`를 확인합니다.
